# виды чатов

- **Дискорд чат**  
  Сообщения отправляются в канал #《🎮》игᴘᴏʙᴏй-чᴀт и видны всем игрокам.

- **Глобальный чат**  
  Сообщения с префиксом **!** видны всем игрокам на сервере.

- **Локальный чат**  
  Сообщения помечаются серым **|** и видны только игрокам в радиусе 200 блоков.

- **Торговый чат**  
  Сообщения с префиксом **$** видны всем игрокам и используются для торговли и обмена.


# настройки чата

<img className="block " src="/images/mechanics/chat_intro.png" alt="Hero Light" class = "rounded-lg" />

Команда `/chat` открывает меню настроек чата, где можно включать или отключать разные типы сообщений и уведомлений.

---

## Основные переключатели

| Опция | Описание |
|-------|----------|
| **Вход/Выход** | Показывать сообщения о входе и выходе других игроков. |
| **Глобальный чат** | Получать сообщения в глобальном чате. |
| **Достижения** | Показывать сообщения о достижениях игроков. |
| **Личные сообщения** | Получать личные сообщения. |
| **Звуки** | Включить звуковые уведомления для чата. |
| **Упоминания** | Получать уведомления при упоминании вашего ника. |
| **Очистка предметов** | Получать уведомления об очистке предметов. |
| **Онлайн** | Получать уведомления об онлайн игроках на сервере. |


## Расширенные возможности (спонсоры и модераторы)

| Опция | Описание |
|-------|----------|
| **Свой вход/выход** | Показывать свои собственные сообщения о входе/выходе. |
| **Своя смерть** | Показывать свои сообщения о смерти. |



## Дополнительно

- **Сброс настроек** — кнопка для возвращения всех переключателей к значениям по умолчанию.

# настройки цвета чата (только для спонсоров)

команда `/chatcolor` позволяет выбрать цвет вашего ника в чате.  
**важно: доступна только игрокам со статусом спонсора.**



## как использовать

### 1-й вариант
1. откройте меню `/chatcolor`.  
2. выберите желаемый цвет.  
3. цвет автоматически применяется ко всем вашим сообщениям в чате.

# команда очистки чата /clearchat

Команда `/clearchat` очищает чат для одного игрока, то есть **только для вас**.  
Она не удаляет сообщения для других игроков — просто позволяет убрать всю историю сообщений со своего экрана.
