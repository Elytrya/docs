> **🖼️ картинки**
> используйте команды ниже, чтобы добавлять в мир собственные изображения по ссылке.

---

## 🛠️ команды

- `/frame create <название> <ссылка> <ширина> <высота>` — создать изображение.
- `/frame delete <название>` — удалить изображение.
- `/frame get <название>` — получить изображение в виде предмета.
