## 📘 Список команд

| Команда | Описание |
|--------|----------|
| `/achievements` | Кастомные достижения |
| `/afk` | Стать в АФК |
| `/chat` | Настройки чата |
| `/clearchat` | Очистить чат для себя |
| `/ignore <игрок>` | Игнорировать игрока |
| `/particles` | Настройки партиклов (если они у вас есть) |
| `/playtime` / `/playtime top` | Общее наигранное время / Топ по наигранному времени |
| `/plew` / `/plew <игрок>` | Плюнуть / Плюнуть в игрока |
| `/seen <игрок>` | Посмотреть, когда игрок был в сети в последний раз |
| `/lore текст` | Добавить подпись предмету |
